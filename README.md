# Heap-Reference-Analysis-in-SLIM

This repository contains the report for a static analysis to optimize the heap memory usage of a program.

This was done as part of a B.Tech. project at IIT Bombay.

Contributions: 
- Defining and implementing the Data Flow Values (DFVs) for the analysis
- Implementing the helper functions for the meet operator and transfers in the analysis
- Designing a data structure for memoizing results computed during the analysis and efficiently handling DFV construction
